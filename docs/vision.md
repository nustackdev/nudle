# nudle — Vision

## What

nudle is the UI for Nu.

Nu is a programmable database where data topology is the program — Shapes declare structure, Terms describe computation. No existing tool can see into it. nudle is the lens.

## How

The UI is a Term builder and a result renderer. Every interaction — navigating, editing, filtering, slicing — constructs a Term. The backend evaluates it. The frontend draws the result. That's the entire loop.

## Who

Three writers share the tree: **human**, **agent**, **machine**. nudle is the human's interface. Everything in the tree is visible and editable — regardless of who wrote it, how it got there, or what substrate it lives on.

## Bar

Delightful UX is not optional. This is not an admin panel. This is how humans experience Nu.
